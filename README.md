# COMP331-Final-Project

This project was done for COMP331 at The University of Fraser Valley. The delivarables include and introduction talking about the case study chosen, whether it was one on your own or the 2 options given, why data quality matters and the quality dimensions chosen. At least 2 data quality dimensions must be chosen and used to interpret the given or chosen data set identifying issues which each as well as connecting learned concepts. Along with this there must be at least 3 recommendations prioritized by feasability and impact and a way to address the issues. Finally a simple conclusion is asked for summarizing key findings, reflecting on lessons as well as connecting learning outcomes to course material. 

A reference page will also be attached to the document and output files come along with the code.
